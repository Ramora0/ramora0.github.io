Research

Dynamic Tokenization for ASR

DeepMind Carnegie Mellon Publication in progress

Developing boundary prediction methods for dynamic tokenization in automatic speech recognition. Instead of fixed-length tokens, the system learns variable-length segments that align with linguistic structure — achieving comparable ASR performance with roughly 5× compression, significantly reducing sequence lengths without sacrificing accuracy.

Searchless Chess via Learned Evaluation

Transformer No search

A transformer-based chess encoder that plays entirely without search — no MCTS, no minimax, no lookahead. The model evaluates positions and selects moves directly from board state, relying on learned pattern recognition rather than tree traversal.
